# Buys Eye

# Project Description
Simple SwiftUI app build in a raywenderlich iOS course.
This project teach me some topics:
  - Custom alert 
  - Gradient backgrounds
  - Dark / Light theme change
  - Navigate to another screen 
  - Use size class Environment property
  - Use color scheme Environment propery
  - Introduction to Unit Tests
  - Separate views
  - Basic animations
